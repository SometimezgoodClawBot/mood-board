# Mood Board (House Colour Consultation)

A single-file, offline-ready mood board web app for organising paint/colour inspirations by room.

## Features

- Single self-contained `index.html` (inline CSS/JS)
- Upload + drag/drop images
- Room organisation: Kitchen, Living Room, Bedroom, Bathroom, Exterior, Other
- Label + notes per image
- Select 2–3 images to compare side-by-side
- Export as PDF via print (Save as PDF)
- Export/import board as JSON
- Persists locally via `localStorage`
- Mobile-responsive UI
- Dark/light/system theme toggle

## Usage

Open `index.html` in any modern browser.

Tips:
- For best performance, use reasonably-sized images.
- PDF export uses the browser print dialog.

## Privacy

All data stays in your browser (localStorage) unless you export it.
